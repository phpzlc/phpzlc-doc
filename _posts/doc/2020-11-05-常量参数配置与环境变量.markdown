## 常量参数配置

配置文件位置：
```
config/services.yaml
```
常量配置：
```yaml
    ##字符串
    var_string: 这是个字符串
    ##数组
    var_array:
        0 : 选项1
        1 : 选项2
```

## 环境变量

配置文件位置
```
.env
```
```env
##字符串
ENV_STRING=这是个字符串
```
**.env文件配置项目默认环境变量，实际开发中，默认环境变量及其配置注释不可忽略，赋予的值可以是写法示例**

本地环境变量位置
```yaml
.env.local
```
**.env.local文件不入git，属于本机环境独有的环境变量，需要手动创建。**
```env
##字符串
ENV_STRING=这是个字符串
```
**.env.local文件配置的环境变量会对应覆盖.env文件中值**
