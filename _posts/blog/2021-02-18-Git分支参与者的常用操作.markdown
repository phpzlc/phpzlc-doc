## 查看分支

```shell
git branch -a
```

## 切换分支

```shell
git checkout beta

git checkout develop
```

## 合并分支

```shell
git merge --no-ff develop

git merge --no-ff beta
```

## 强制更新分支到本地

```shell
git fetch origin master

git reset --hard origin/master
```

## 清除git缓存提交


```shell
git rm -r --cached .

git add .

git commit -m 'update .gitignore'

git push
```
